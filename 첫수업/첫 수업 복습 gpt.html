첫 수업때 배운 것: Git과 GitHub

우선 Git에 대해 배운 내용을 정리해보겠습니다.

Git은 폴더나 파일의 타임머신 장치입니다.

GitHub는 이 타임머신의 저장소를 관리하는 사이트입니다.

Git을 시작하려면 먼저 git init 명령어를 사용하여 Git을 시작할 폴더를 지정해야 합니다.

그 후, git add . 명령어를 사용하여 현재 폴더를 Git이 관리하는 폴더로 지정합니다.

그리고 git commit -m "메시지" 명령어를 사용하여 현재 폴더의 상태를 저장합니다.

그 전에, git config --global user.name "사용자명"과 git config --global user.email "이메일" 명령어로 사용자 정보를 설정합니다.

GitHub에서 새로운 저장소를 만들고, 링크를 복사한 후 Git에 붙여넣습니다. 이때 Ctrl + V 대신 마우스 오른쪽 클릭 후 Paste를 사용합니다.

그리고 git push -u origin master 또는 git push -u origin coffee 명령어로 GitHub에 올릴 수 있습니다.
