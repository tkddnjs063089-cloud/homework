우선 git 배운거 부터
git은 폴더나 파일의 타임머신 장치이다
github는 타임머신의 저장소를 관리하는 사이트 이다
git을 시작 하려면 먼저 git init이라는 명령어를 사용하여 git을 시작할 폴더를 지정해 주어야 한다
그리고 git add . 이라는 명령어를 사용하여 현재 폴더를 git이 관리하는 폴더로 지정해 준다
그 다음 git commit -m "메시지" 라는 명령어를 사용하여 현재 폴더의 상태를 저장한다
그 전에 git config --global user.name "사용자명"과 git config --global user.email "이메일" 명령어를 사용하여 사용자 정보를 설정 해 준다.
그리고 githun에서 새로운 저장소를 만들어 링크를 복사 후 git 에 붙여넣는다. 이떄 ctrl + v 는 사용하지 말고 마우스 우클릭 후 paste를 사용한다.
그리고 git push -u origin 'master,coffee' 등등 을 기입하여 github에 올릴 수 있다.
